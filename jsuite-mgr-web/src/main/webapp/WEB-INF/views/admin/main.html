jsuite后台管理框架介绍

jsuite是一个企业级快速开发平台，也是一个基于业务的业务组件框架,提供业务管理及api接口调用， 便于二次开发及集成，它可以用于所有的Web应用程序，如网站管理后台， 网站会员中心，CMS，CRM，OA等等， 也可以对它进行深度定制，以做出更强系统。

当前版本：1.0

源码 访问主页

源码工程介绍：

  • jsuite-article：文章管理
  • jsuite-common：公共管理，包括：地区管理
  • jsuite-log:日志模块，提供内部操作日志相关的拦截
  • jsuite-user：用户模块，主要是系统后台管理模块,提供首页菜单、用户管理、菜单管理、角色管理。
  • jsuite-file-pom：文件上传,支持本地文件上传、阿里云oss文件上传
  • jsuite-generator：代码生成器。
  • jsuite-mgr-web：管理应用启动工程示例使用freemarker
    •

    技术选型：

  • 核心框架：rabbitframework
  • spring框架：Spring Boot。
  • 安全框架：Apache Shiro。
  • 模板引擎：freemarker。
  • 数据库连接池：Druid。
  • 工具类：Fastjson。
  • 前端：jquery、hplus等。
